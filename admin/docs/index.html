Welcome to Joomla 3.0

Dozens of reusable components are built from Bootstrap to provide navigation, alerts, popovers, and much more. This is what we think Joomla can become. With support we can make this happen quickly.

Strategy Lets do this together

The Problem

The community of Joomla developers have always struggled with the tools they have been given in the Joomla API, and end up innovating outside the core, essentially creating their own UI kits. The result is the current disjointed, nonstandard set of component interfaces that all have different design patterns and usability issues.

The Vision

The vision of the JUX will improve the daily lives of all Joomla users: designers, developers (core and community), site administrators.

With proper standards and a set of UI tools, designers can more rapidly prototype wireframes and flows.

The Framework

For Joomla 3.0 we'd like to use the Bootstrap Framework. This comes with the majority of css, html, and javascript that any developer or designer would need.

Of course we know this will not allow for all cases but Bootstrap gives you the ability to customize as you see fit.

The Pros

3rd party developers will have this api at their fingertips. What they once didn't have designers, they no longer will need. This will unify the look of Joomla and it's extensions and give developers the freedom to no longer need to make their own cusom UI.

The Cons

Some developers may feel locked into this UI, but for most it's a way to develop rapidly. If you still don't want to follow the UI standards you don't have to.

The Future

This will set Joomla ahead of the curve, and send it using lightspeed. Where other CMS have just started working on UX, Joomla will already be ahead.
